# VIEW-見るだけ

参照専用・読み取り専用フォルダです。

## フォルダ構成

### documentation/
- プロジェクトドキュメント
- 仕様書・設計書
- ガイドライン・マニュアル

### examples/
- サンプルコード・デモ
- 使用例・実装例
- チュートリアル用コード

### reference/
- リファレンス資料
- API仕様書
- 外部資料・リンク集

### reports/
- レポート・ログ
- 分析結果・調査報告
- 作業履歴・進捗報告

## 使用方法

1. 情報収集・学習時に参照
2. **編集・変更は行わない**（読み取り専用）
3. 新しい資料は適切なサブフォルダに追加
4. 定期的に内容を整理・更新

## 注意事項

- このフォルダ内のファイルは参照用です
- 編集が必要な場合はCHANGE-変更するフォルダにコピーしてから作業
- 重要な情報の削除は慎重に行う
