## 🌈 評価軸の定義（各10点満点）

- **技術実現性**: 技術的制約下での実現可能性
- **効率性**: 時間・リソースの最適化度
- **生産性**: 開発効率向上への寄与度
- **保守性**: 長期メンテナンスの容易さ

- **ユーザビリティ**: 使いやすさ・理解しやすさ
- **リスク管理**: 失敗防止・対応策の充実度
- **革新性**: 既存手法からの創造的突破
- **独自性**: 他では見られない固有のアイデア
